# SpaceCraft Mission Terminal

A retro-inspired spacecraft mission terminal built with **HTML5 Canvas**, **CSS3**, and **Vanilla JavaScript**. The project simulates a futuristic mission control interface featuring animated scanlines, glowing CRT effects, a blinking cursor, and a dynamic fuel status indicator.

## Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Styling and layout |
| Vanilla JavaScript (ES6) | Application logic |
| HTML5 Canvas API | Rendering the interface |
| Canvas 2D Context | Drawing graphics |
| requestAnimationFrame() | Smooth animation loop |
| CSS Flexbox | Centering the canvas |

## ⚙️ Installation

1. Clone the repository.

```bash
git clone https://github.com/Zunairatahir/spacecraft-mission-terminal.git
```

2. Navigate to the project folder.

```bash
cd spacecraft-mission-terminal
```

3. Open `index.html` in your preferred web browser.

No additional dependencies or installations are required.
